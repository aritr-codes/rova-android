# Phase 6.1b — Dual-recording consumer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Activate the P+L Mode tab by wiring the Phase 6.1a `service.dualrecord` package into `RovaRecordingService`, extending manifest schema 4→5 with additive `SegmentRecord.side`, threading mode-awareness through storage / recovery / export, and adding a Compose split-preview overlay — so a P+L session produces two playable clips per ordinal, two merged outputs per session, two MediaStore entries, and two History cards.

**Architecture:** Consumer wire-up of 6.1a's foundation. `setupCamera()` becomes a 1-line dispatcher to `setupSingleCamera()` (verbatim extract) vs `setupDualCamera()` (new). Manifest schema 4→5 with additive `SegmentRecord.side: VideoSide?` (null = legacy/single-mode). Recovery scanner regex extends to optional `_P`/`_L` suffix; grouping shifts from `(index)` to `(index, side)`. Storage gate gains mode parameter cascade. ExportPipeline gains side parameter; SessionStore gains per-side export mutators; `performMerge` orchestrates 2 ExportPipeline calls + single terminal write for P+L. Compose split overlay (informational, not cropped) lives inline in `RecordScreen`.

**Tech Stack:** Kotlin 2.2.10 · AGP 9.2.1 · Gradle 9.4.1 · compileSdk / targetSdk 37 · CameraX 1.4.2 · JUnit + Mockito-Kotlin for pure-JVM tests. No Robolectric. No new dependencies.

**Branch:** `feat/dual-recording-consumer` already created from `master @ 5224e83`. Spec committed at `151212a`.

**Spec:** `docs/superpowers/specs/2026-05-15-phase-6.1b-dual-recording-consumer-design.md` (D1–D12 + 9 Open Qs).

private suspend fun performMergeDual(
    portraitSegments: List<File>,
    landscapeSegments: List<File>
) {
    var portraitOk = false
    var landscapeOk = false
    try {
        _serviceState.update { it.copy(isMerging = true, mergeProgress = 0f, mergeError = null) }
        val totalSegments = portraitSegments.size + landscapeSegments.size
        updateNotification(NotificationState.Merging(done = 0, total = totalSegments))

        val sid = currentSessionId
        val sessionDir = currentSessionDir
        if (sid == null || sessionDir == null) {
            RovaLog.w("performMergeDual: no active session; skipping export")
            _serviceState.update { it.copy(mergeError = "No active session") }
            return
        }
        if (!sessionDir.exists()) sessionDir.mkdirs()

        // Portrait pass
        if (portraitSegments.isNotEmpty()) {
            val portraitResult = ExportPipeline.export(
                context = this@RovaRecordingService,
                sessionStore = sessionStore,
                sessionId = sid,
                sessionDir = sessionDir,
                segments = portraitSegments,
                side = com.aritr.rova.service.dualrecord.VideoSide.PORTRAIT
            ) { progress ->
                _serviceState.update { it.copy(mergeProgress = progress * 0.5f) }   // first half
            }
            portraitOk = portraitResult is ExportResult.Success
            if (!portraitOk) RovaLog.e("performMergeDual: portrait export failed: $portraitResult")
        }

        // Landscape pass — runs even if portrait failed (independent atomicity per D12)
        if (landscapeSegments.isNotEmpty()) {
            val landscapeResult = ExportPipeline.export(
                context = this@RovaRecordingService,
                sessionStore = sessionStore,
                sessionId = sid,
                sessionDir = sessionDir,
                segments = landscapeSegments,
                side = com.aritr.rova.service.dualrecord.VideoSide.LANDSCAPE
            ) { progress ->
                _serviceState.update { it.copy(mergeProgress = 0.5f + progress * 0.5f) }   // second half
            }
            landscapeOk = landscapeResult is ExportResult.Success
            if (!landscapeOk) RovaLog.e("performMergeDual: landscape export failed: $landscapeResult")
        }

        // Cleanup successful-side raw segments
        withContext(Dispatchers.IO) {
            if (portraitOk) {
                portraitSegments.forEach { try { it.delete() } catch (_: Exception) {} }
            }
            if (landscapeOk) {
                landscapeSegments.forEach { try { it.delete() } catch (_: Exception) {} }
            }
        }

        if (portraitOk || landscapeOk) {
            // At least one side succeeded — write terminal as the single-mode path does
            updateNotification(NotificationState.MergeComplete(clipCount = totalSegments))
            delay(1000)
        } else {
            // Both sides failed
            _serviceState.update { it.copy(mergeError = "Both sides failed") }
            updateNotification("Merge failed")
            delay(3000)
        }

    } catch (ce: CancellationException) {
        throw ce
    } catch (e: Exception) {
        e.printStackTrace()
        _serviceState.update { it.copy(mergeError = e.message) }
        updateNotification("Merge failed: ${e.message}")
        delay(3000)

        _serviceState.update { it.copy(isMerging = false) }
        // Single terminal write across both sides per ADR 0006 §"Terminal-Write Ordering"
        if (portraitOk || landscapeOk) {
            // … markTerminated logic — mirror the existing performMerge terminal-write block.
