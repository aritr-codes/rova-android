The rotate model: the bottom cluster (config strip + Library/REC/Settings + sheet) is glued to the physical old-bottom edge — right in 90°, left in 270° — with order following the rotation (reversed in A, identity in B), exactly like the native camera's icon column. Nav stays edge-most, config stays inboard (same depth order as portrait). Status + cam utilities are screen-anchored to the top corners, as in native. All text upright (WCAG — our one divergence from Samsung's sideways labels). Use CONFIG WEIGHT to compare the proposed slim chips against the rejected Phase-A weight — geometry identical, only density changes.
